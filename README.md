# A Shoes Store REST API built with GoLang
# Technologies
* Go
* [Gorm](https://gorm.io/index.html). (ORM library for Golang)
* [Fiber](https://docs.gofiber.io/)
* Database: MySQL
# Key Features
* Product CRUD
* Category CRUD
* Post CRUD
* Create order
* Authentication (JWT, Cookies,...)
